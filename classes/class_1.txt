Class: Hamburger

Attributes:
* patties (integer)
* cheese (boolean)
* internalTemp (string)
* bun (string)

Method:
* pattyCount (returns a number for the number of patties based on user selection. This modifies the patties attribute)
* addCheese (boolean that adds cheese to the hamburger if true. This modifies the cheese attribute)
* changeCookTemp (modifies the cook temperature of the hamburger. Such as Rare, Medium-Rare, Medium, etc. This modifies the internalTemp attribute)
* selectBun (changes the type of bun used on the hamburger. Such as plain or sesame bun. This modifies the bun attribute)
